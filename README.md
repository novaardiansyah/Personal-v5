# Personal Filament v5 Admin Panel

Hi Future Developer,

Welcome to the **Personal Filament v5 Admin Panel**! This project is an open-source admin panel built with Laravel 13 and Filament PHP 5, designed for personal utility management and more.

This repository is the next evolution of my previous admin panel project. Built from the ground up with Laravel 13, PHP 8.5, Filament PHP v5, and PostgreSQL taking advantage of the latest ecosystem improvements for performance, developer experience, and scalability.

> **Note:** This project was rewritten from the previous version (personal-v4) which used Laravel 12, PHP 8.2, and MySQL.

## Dazzling Tech Stack

- [Laravel 13](https://laravel.com/) - Backend PHP Framework
- [Filament PHP 5](https://filamentphp.com/) - TALL Stack Admin Panel
- [PHP 8.5](https://www.php.net/) - Server-side language
- [PostgreSQL](https://www.postgresql.org/) - Database Management
- [Tailwind CSS 4](https://tailwindcss.com/) - CSS Framework
- [Livewire 4](https://livewire.laravel.com/) - Full-stack framework for Laravel
- [Pest 4](https://pestphp.com/) - Testing Framework
